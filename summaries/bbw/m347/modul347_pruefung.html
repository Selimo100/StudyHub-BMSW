D1FHardOffene Frage
Erstelle eine docker-compose.yml für NGINX mit: Service web · Image nginx:alpine · Containername d1f-web · Port 8080→80 · Automatischer Neustart (ausser manuell gestoppt)
0 Zeichen
✏️ Musterlösung
services:
  web:
    image: nginx:alpine
    container_name: d1f-web
    ports:
      - "8080:80"
    restart: unless-stopped

E1FMediumOffene Frage
Cloud-Instanz: Ubuntu 24.04, öffentliche IP, Firewall: Port 22 und 80 offen.
Erkläre die Funktion von Port 22 und Port 80 und warum diese Konfiguration für einen Webserver sinnvoll ist.
0 Zeichen
✏️ Musterlösung
Port 22 (SSH): Sicherer Remote-Zugriff zur Administration der Cloud-Instanz.
Port 80 (HTTP): Zugriff auf den Webdienst über den Browser.

Warum sinnvoll: Die öffentliche IP erlaubt externen Zugriff. Die Firewall öffnet nur die benötigten Ports → erhöhte Sicherheit.
