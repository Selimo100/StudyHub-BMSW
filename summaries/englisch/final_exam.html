Quick Rules and Common Traps

Use this as a last-minute checklist.

Tense Signals

Tense Signal words
Present Simple always, often, every day
Present Continuous now, at the moment, today
Present Perfect ever, never, already, yet, since, for
Past Simple yesterday, ago, last week
Past Continuous while, when
Past Perfect before, after, by the time

Conditionals

Type Structure
0 If + present, present
I If + present, will + base verb
II If + past, would + base verb
III If + past perfect, would have + past participle
Never write: If it will rain. Write: If it rains.

Verb Pattern Traps

enjoy/avoid/finish/suggest + -ing. want/decide/hope/promise + to. tell/ask/remind + object + to. make/let/help + object + base verb. After prepositions, always use -ing.

Remember to lock = future duty. Remember locking = memory.
Stop smoking = quit. Stop to smoke = pause for another action.

Passive Forms

is done, was done, has been done, had been done, will be done, is going to be done, is being done, must be done.

Question Traps

After do/does/did, use the base verb. Subject question: Who called you? Object question: Who did you call?

Vocabulary Traps

Use: do homework, make a mistake, take a photo. Information, advice, money and homework are uncountable.
